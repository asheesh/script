# Kubernetes Lens

## Run the following script to install Lens on your System

Steps:- sudo bash installLensApp.sh

1. This will install the binary on your system in /usr/local/bin

2. Start your Lens by running the binary   Lens-3.5.0.AppImage

3. You need to give your kubeconfig file so that lens can setup the clusters

4. You are now ready to use it  Enjoy the Dashboard
